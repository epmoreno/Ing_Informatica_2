\doxysubsection{Descripción detallada}
Representa una colección de fechas históricas. 

La clase \doxylink{classCronologia}{Cronologia} permite almacenar, buscar y manipular un conjunto de fechas históricas. Cada fecha histórica es un objeto de la clase \doxylink{classFechaHistorica}{Fecha\+Historica}, que contiene un año y una lista de eventos que ocurrieron en ese año. La cronología almacena las fechas históricas en un vector.

{\footnotesize\ttfamily std\+::string Cronologia\+::to\+String (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const}



Convierte la cronología a una cadena de texto. 

Esta función es responsable de convertir una cronología completa a una representación de cadena de texto. La cadena de texto resultante tiene el formato "{}\+Fecha\+Historica1\textbackslash{}n\+Fecha\+Historica2\textbackslash{}n..."{}.(Siendo \textquotesingle{}~\newline
\textquotesingle{} un salto de línea).

Primero, la función inicializa una cadena de texto vacía llamada {\ttfamily resultado}. Luego, la función recorre cada fecha histórica en el vector de fechas históricas. Para cada fecha histórica, la función convierte la fecha histórica a una cadena de texto llamando a la función {\ttfamily to\+String} de la fecha histórica. Luego, añade la cadena de texto de la fecha histórica a {\ttfamily resultado}, seguida de un carácter de nueva línea. Después de recorrer todas las fechas históricas, la función devuelve {\ttfamily resultado}.

\begin{DoxyReturn}{Devuelve}
Devuelve una cadena de texto que representa la cronología. ~\newline

\end{DoxyReturn}
\begin{DoxyNote}{Nota}
Esta función no modifica el objeto \doxylink{classCronologia}{Cronologia}. Solo accede a los datos del objeto y los utiliza para crear una nueva cadena de texto. 
\end{DoxyNote}
